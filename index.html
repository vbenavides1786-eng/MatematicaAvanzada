Crear un dataset a partir de datos estadsticos

en muchas ocasiones encontramos bibliografa que nos trae como nico dato de un dataset lo que se conoce como valores paramtricos. Imaginemos que en un artculo cientfico se nos dice que el tiempo medio entre fallos de un motor elctrico es de 196 das con un desvio estandar de 60. Crearemos un conjunto de datos para hacer simulaciones con estos dos parmetros

mtbf = rnorm(500,196,60)

Obtener elementos de una lista

Para obtener elementos de juna lista se usa el comando mtbf[a], donde a es el nmero del elemento a seleccionar de la lista. En caso se requiera un conjunto seleccionado de elementos a= 125:130, es decir del elemento 125 a 130.

Como introducir valores ausentes en nuestro conjunto de datos

mtbf[indices] <- NA
